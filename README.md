# Finance_Dashboard (Interactive Dashboard creation using Power BI)
## project Objective
This project visualizes key financial insights from a structured dataset containing:

- Customer details
- Transaction details
- Branch information
- State & city data
- Payment modes
- Transaction types

The dashboard helps organizations monitor revenue, trends, customer activities, and operational performance.

## Project Insights

- Credit transactions dominate, showing high digital payment usage.
- Metro and Main branches generate the highest transaction values.
- A few states and major cities contribute most of the financial activity.
- Monthly trends show clear peaks during salary periods and festive seasons.

## Final Conclusion

The Finance Dashboard provides a clear view of financial activity across states, branches, and customer groups. It highlights digital payment growth, strong metro performance, and seasonal transaction surges. These insights enable faster and more accurate data-driven decisions for financial planning and operations.
