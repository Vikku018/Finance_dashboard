# Finance_Dashboard (Interactive Dashboard creation using Power BI)
## project Objective
This project visualizes key financial insights from a structured dataset containing:

- Customer details
- Transaction details
- Branch information
- State & city data
- Payment modes
- Transaction types

The dashboard helps organizations monitor revenue, trends, customer activities, and operational performance.

## Questions

### (KPI card)
- Total Transactions
- Avg Transaction
- Total Credits
- Total Debits
### DAshboard Question
- share of each payment mode in total transactions?
- transaction amount vary across different transaction types?
- monthly transaction trends over time?
- cities contribute the highest transaction amount?
